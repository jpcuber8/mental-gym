# App Flow

## Primary Daily Flow

Mental Gym is primarily a pre-run app. The normal use case is to open it before the run, workout, warm-up, race routine, travel-day movement, or recovery block.

1. Open app before training.
2. Review today's date, current phase, next key event, and recommended session.
3. Start the daily pre-run mental workout.
4. Complete readiness check.
5. Complete regulation primer.
6. Complete primary skill rep.
7. Complete XC transfer rep.
8. Save reflection, usefulness, execution, and carryover cue for the run.
9. Review progress or leave.

## Default Session Architecture

1. Readiness check, 20-40 seconds.
   - stress/arousal, 0-10
   - focus, 0-10
   - confidence, 0-10
   - mental freshness, 0-10
   - optional notes
   - upcoming run/workout type

2. Regulation primer, 60-120 seconds.
   - usually 4-second inhale, 6-second exhale
   - calm, low-friction, not fast breathwork

3. Primary skill rep, 2-4 minutes.
   - breathing
   - attention anchor
   - self-talk ladder
   - imagery
   - process goal
   - if-then planning
   - routine rehearsal
   - reset/refocus for residue carried into the next run

4. XC transfer rep, 60-120 seconds.
   - pack running
   - hills
   - surges
   - discomfort in intervals
   - closing hard
   - getting boxed in
   - first kilometer patience
   - altitude camp adjustment
   - race-morning nerves
   - next-run reset after a rough workout or race
   - transition from summer to UCLA/team environment

5. Reflection, 20-30 seconds.
   - usefulness, 0-10
   - execution, 0-10
   - cue for next run
   - optional note

## Main Sections

### Today

- Date.
- Current season phase.
- Days until next key event.
- Recommended daily pre-run session.
- Start session button.
- Weekly sessions completed.
- Recent stress, focus, confidence, freshness.
- Why today explanation.

### Daily Session

- Step-by-step guided flow.
- Progress indicator.
- Timers where useful.
- Slider-first input to avoid typing.
- Reflection saved at the end.

### Progress

- Weekly adherence.
- Stress, focus, confidence, freshness trends.
- Cue history.
- Session history.

### Season Plan

- Timeline from May 22 through November 13.
- Highlights London travel, 10-mile race, UCLA reporting, Big Bear, Nuttycombe, Big Tens, and West Regional.
- Shows mental priority for each phase.

### Stations

- Breathing.
- Attention.
- Self-talk.
- Imagery.
- Process goals.
- Routine rehearsal.
- Reset/refocus.

Each station includes explanation, levels 1-4, XC examples, and manual start.
